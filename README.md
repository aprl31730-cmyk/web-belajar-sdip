# Web Belajar SDIP Al Madinah

## Cara menjalankan lokal
1. Install Node.js
2. Buka terminal di folder ini
3. Jalankan:
   npm install
   npm run dev

## Deploy ke Vercel
- Upload folder ini ke GitHub
- Import ke Vercel
- Deploy

## Firebase
- Isi file src/firebase.js dengan config dari Firebase Console
